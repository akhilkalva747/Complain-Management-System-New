For a fresh laptop — do this in order:
1. Copy the project folder to the new laptop (USB/zip). Put it somewhere without & in the path, e.g. C:\CMS\.

2. Double-click setup.bat — it automatically:

Installs Java 21 (via Windows winget, built into Win 10/11)
Downloads & extracts Maven 3.9.6 to C:\cms-tools\maven
Installs Node.js LTS (via winget)
Installs XAMPP (MySQL) via winget
Creates the querysolve database and imports all sample data
Runs npm install for the frontend
3. Open a NEW CMD window (so PATH refreshes with Java/Node)

4. Double-click start-app.bat — opens 3 windows automatically:

MySQL (XAMPP)
Spring Boot backend on port 8080
React frontend on port 3000 (browser opens automatically)

Note: If winget XAMPP install fails (common on some machines), install XAMPP manually from https://www.apachefriends.org/ then run setup.bat again — it will skip already-installed tools.
